<h1>Sistema de leilões em java </h1>

Projeto desenvolvido em Java utilizando Swing para a interface gráfica e MySQL para persistência de dados, seguindo o padrão 
de arquitetura DAO (Data Access Object).
O sistema permite o gerenciamento de produtos de um leilão, possibilitando o cadastro, consulta e atualização do status dos itens.

Funcionalidades: 

- Cadastro de produtos
- Listagem de produtos cadastrados
- Integração com banco de dados MySQL via JDBC
- Arquitetura organizada em DAO e DTO
- Interface gráfica desenvolvida com Java Swing
- Estrutura preparada para implementação de venda de produtos e consulta de vendas

Tecnologias utilizadas:

- Java
- Java Swing
- JDBC
- MySQL
- NetBeans

Estrutura do projeto: 

DTO – Responsável pelo transporte dos dados dos produtos.
DAO – Responsável pelas operações de acesso ao banco de dados.
VIEW – Interfaces gráficas do sistema.
JDBC – Conexão entre a aplicação e o banco de dados.

Objetivo: 

Este projeto foi desenvolvido com fins acadêmicos para praticar conceitos fundamentais de desenvolvimento Java, incluindo 
programação orientada a objetos, arquitetura em camadas, integração com banco de dados relacional e construção de interfaces desktop.

Funcionalidades previstas: 

- Cadastro de produtos
- Consulta de produtos
- Venda de produtos
- Histórico de vendas
- Atualização automática do status dos produtos
